Contenedores

En CSS, un contenedor es un elemento que contiene uno o más elementos secundarios. Un contenedor suele ser un elemento div o de sección y se utiliza para agrupar y organizar elementos relacionados en una página web. Un contenedor puede tener un diseño flexible o un diseño de cuadrícula para que los elementos secundarios se dispongan en un orden determinado. Las propiedades del contenedor, como la visualización, el contenido de la justificación y el elemento de alineación, se utilizan para establecer el comportamiento y la apariencia del contenedor y sus elementos secundarios.

Las clases que puedes usar para cada contenedor son:
  • .container
  • .container-fluid
  • .container-sm
  • .container-md
  • .container-lg
  • .container-xl
  • .container-xxl

Código base:

.container, .container-fluid, .container-sm, .container-md, .container-lg, .container-xl, .container-xxl {

width: 100%;

padding-right: 15px;

padding-left: 15px;

margin-right: auto;

margin-left: auto;

}

Tipos de contenedores:

Contenedor small para teléfonos móviles pequeños

@media (min-width: 576px) {

.container, .container-sm{

max-width: 556px;

}

}

Contenedor small para teléfonos móviles medianos

@media (min-width: 768px) {

.container, .container-sm, .container-md{

max-width: 740px;

}

}

Contenedor small para tablet

@media (min-width: 992px) {

.container, .container-sm{

max-width: 970px;

}

}

Contenedor small para laptops

@media (min-width: 1200px) {

.container, .container-sm, .container-md{

max-width: 1180px;

}

}

Contenedor small para monitores anchos

@media (min-width: 1400px) {

.container, .container-sm, .container-md{
