Alerts

Las alertas permiten notificar sobre algo al usuario. Ejemplos:

warning-alert

.warning-alert{

@extend .generic-style;

background-color: $blue-color;

font-family: $font-1;

font-size: $size-3;

color: $white-colour;

}

successfully-alert

.successfully-alert{

@extend .generic-style;

background-color: $green-colour;

color: $white-colour;

font-family: $font-1;

font-size: $size-3;

}

error-alert

.error-alert{

@extend .generic-style;

background-color: $yellow-colour;

color: $black-colour;

font-family: $font-1;

font-size: $size-3;

}

danger-alert

.danger-alert{

@extend .generic-style;

background-color: $red-colour;

color: $white-colour;

font-family: $font-1;

font-size: $size-3;

}
