Badges

Un badge (distintivo o insignia) es una pequeña etiqueta que se agrega a otras componentes para mostrar un pequeño mensaje o contador.

Esta es una Notificación

.badge, .badge-icono {

display: inline-block;
padding: 0.3rem;
font-size: 0.75rem;
font-weight: 600;
line-height: 1;
color: white;
vertical-align: baseline;
text-align: center;
white-space: nowrap;
border-radius: 0.15rem;

}

Para darle ese estilo tan bonito, agregamos, en este caso, un span y le colocamos las clases "badges" y un color de nuestra preferencia.

.badge, .badge-icono {

display: inline-block;
padding: 0.3rem;
font-size: 0.75rem;
font-weight: 600;
line-height: 1;
color: white;
vertical-align: baseline;
text-align: center;
white-space: nowrap;
border-radius: 0.15rem;

}

.badge-icono{

position: absolute;
transform: translate(-50%, -50%);
border-radius: 50rem;

}

Sombras

Con HTML5 podemos crear de una manera sencilla tanto sombras para zonas del HTML. Aunque cada uno de estos tipos de sombra tiene su propia etiqueta, la sintaxis sigue la misma lógica.

Sombras Pequeñas: shadow-small

.shadow-small{

padding: $size-5;
margin: $size-1;
background-color: white;
border-radius: 0.25rem;
box-shadow: 0 0.125rem 0.25rem rgba(0, 0, 0, 0.07);

}

Sombras media: shadow-medium

.shadow-medium{

box-shadow: 0 0.5rem 1rem rgba(0, 0, 0, 0.15);

}

Sombras Grandes: shadow-larger
Botón de contorno verde

.shadow-larges{

box-shadow: 5px 5px 20px rgba(0, 0, 0, 0.4);

}
