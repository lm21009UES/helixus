Botones

Los botones son una manera interactiva de manipular el contenido. El botón por default en HTML se ve de la siguiente manera:

Código:

.button-test {

display: grid;

place-items: center;

}

Agregar color a un botón es sencillo. Para ello, simplemente agregamos el nombre de la clase del botón con el color que nos interesa. Las clases de colores disponibles son las siguientes:

Para el color de fuente:

set-blue-font-colour

set-yellow-font-colour

set-green-font-colour

set-lightblue-font-colour

set-black-font-colour

set-red-font-colour

set-white-font-colour

Código:

.set-blue-font-colour{

color: $blue-color;

}

.set-yellow-font-colour{

color: $yellow-colour;

}

.set-green-font-colour{

color: $green-colour;

}

.set-lightblue-font-colour{

color: $light-blue-colour;

}

.set-black-font-colour{

color: $black-colour;

}

.set-red-font-colour{

color: $red-colour;

}

.set-white-font-colour{

color: $white-colour;

}

Colores

$blue-color: #5E81AC;

$yellow-colour: #EBCB8B;

$green-colour: #A3BE8C;

$light-blue-colour: #88C0D0;

$black-colour: #2E3440;

$red-colour: #BF616A;

$white-colour: #ECEFF4;
